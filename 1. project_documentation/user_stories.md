# User Stories & Acceptance Criteria

## Epic 1: User Onboarding

### Story 1.1: As a new user, I want to complete a health profile
**Acceptance Criteria:**
- [ ] User can access onboarding quiz
- [ ] Quiz captures health goals, allergies, preferences
- [ ] Profile is saved and can be edited later
- [ ] Progress is auto-saved during quiz

### Story 1.2: As a user, I want to set my budget constraints
**Acceptance Criteria:**
- [ ] User can select budget range (low/medium/high)
- [ ] Meal plans respect budget constraints
- [ ] Shopping lists show total estimated cost

## Epic 2: Meal Planning

### Story 2.1: As a user, I want personalized weekly meal plans
**Acceptance Criteria:**
- [ ] AI generates 7-day meal plan based on profile
- [ ] Plans use locally available ingredients
- [ ] Recipes include preparation time and difficulty
- [ ] User can regenerate or customize plans

## Epic 3: Shopping & Vendors

### Story 3.1: As a user, I want an optimized shopping list
**Acceptance Criteria:**
- [ ] List aggregates ingredients from meal plan
- [ ] Items grouped by category/vendor
- [ ] Total cost estimation provided
- [ ] List can be shared or exported

## Epic 4: Progress Tracking

### Story 4.1: As a user, I want to track my health progress
**Acceptance Criteria:**
- [ ] User can log daily meals and water intake
- [ ] Visual progress charts available
- [ ] Achievement badges for milestones
- [ ] Weekly progress reports generated
